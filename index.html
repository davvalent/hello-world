Turtl en résumé

Turtl se présente comme une solution de remplacement à Evernote.

  • Point fort : logiciel libre très sécuritaire avec gestion décentralisée des données utilisateur.
  • Point faible : logiciel en développement qui tarde à atteindre un stade de maturité intéressant.

Notes

  • On peut installer Turtl sur son propre serveur pour plus de sécurité. Donc s'adresse aussi à un public plus avisé (pour la section ...).
  • Dernière version majeure : 0.7, octobre 2018. Quelques versions mineures ont été publiées depuis (par exemple, je suis sur la version 0.7.2.5 sur Windows 10).
  • Terminologie particulière pour « étiquetage » : « tableau » au lieu de dossiers ou étiquettes, similaire à Trello ou Kanban(?). Espaces.
  • Fonctionne sur un modèle décentralisé, à l'image d'Internet : serveurs dispersés sur le réseau : les utilisateurs doivent choisir une installation et ne pourront retrouver leur notes qu'en se connectant à cette installation.
    • L'espace disponible sur un serveur pour chaque compte utilisateur peut être différent d'une installation à l'autre.
  • https://turtlapp.com/features/: très intéressant!
  • Les notes se présentent sous forme cartes. Niveaux : espaces dans lesquels ou peut organiser les notes dans des tableaux. Puis on peut également effectuer un classement par thématiques traversales grâce aux étiquettes.

Ce qu'il manque

  • Possibilité de zoom
  • Efface un tablea efface les notes qui sont dedans...
  • Impossiblité de modifier les tableaux et les étiquettes sans activer le mode édition.
